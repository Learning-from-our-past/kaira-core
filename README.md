# ![alt text](http://i.imgur.com/vBIAv3m.png "Kaira logo") Kaira-core

Main module containing logic for data extraction and command line interface.

## Dependencies
* Mongodb 3.4.9
* Python 3

## Setup
```
make create-venv
source kaira-venv/bin/activate
make setup
make test
```
